# 前端项目启动说明

## 启动命令

```bash
yarn install
yarn run dev
```

或使用npm：
```bash
npm install
npm run dev
```

## 端口配置

端口在 `vite.config.js` 中配置，默认80。如80被占用，Vite会自动使用81、82等端口。

## 访问地址

启动成功后访问终端显示的地址，通常是 `http://localhost:80` 或 `http://localhost:81`。

## 常见问题

**依赖安装失败：**
```bash
Remove-Item -Recurse -Force node_modules
yarn install
```

**页面白屏：**
- 按 `Ctrl + Shift + R` 强制刷新
- 检查浏览器控制台（F12）错误信息
